My job at Educabilia

Whiteboard + Sketching + Illustrator/Photoshop + Sublime Text

We use the SCRUM framework, splitting our tasks into sprints. We had daily design meetings where we started sketching on a whiteboard for each task. This allowed us to work together, designing a consistent user flow.

When we reached a consensus, I moved on to design the wireframes in Illustrator. Once our own set of UI components were created, we usually sketched it out and then jumped directly to the browser.

We used semantic HTML5 for the markup, Sass + Compass for the visual styling, We also based our CSS architecture on concepts from SMACSS.
