1.Task Creation

Empty states are a great way to guide the user through the first times he performs an action.
Referring to the user by his name is an nice way to make him feel welcomed.

2. Creating task

In here will title the task, add a category, set a delivery date, assign a responsible person for the task and write a description.

3. Choosing category

A dialog was chosen over a dropdown menu as it allows to show richer content, standarize the pattern with the one used in “Choose task supervisor”.

4. Choosing
delivery date

Uses Android default datepicker.

5. Choosing one or more attachements.

Wireframes, flows, screenshots, documents, spreadsheets,etc...

6. Choosing task supervisor

User is asked to choose a task supervisor among the members of the project or to add a new one by writing his email.

7. Just one more step...

User then clicks in the Save button located at the top and the Task is finally created.
